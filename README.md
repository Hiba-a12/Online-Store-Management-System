# Online Store Management System

This is a Laravel-based web project developed for the Advanced Web Programming course.  
The project represents a simple online store management system with front pages and an admin dashboard.

## Project Description

The Online Store Management System allows users to browse products, view a shopping cart page, contact the store, and access an admin dashboard.  
The project is built using Laravel MVC structure with routes, controllers, and Blade views.

## Features

- Home page
- Products page
- Shopping cart page
- Contact page
- Admin dashboard
- Product management interface
- Order management interface
- Basic store statistics
- Laravel MVC structure
- Blade template pages
- GitHub version control with commit history

## Pages

### Front Website

- `/` : Home page
- `/products` : Products page
- `/cart` : Shopping cart page
- `/contact` : Contact page

### Admin Panel

- `/admin` : Admin dashboard page
